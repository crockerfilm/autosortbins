# AutoSortBins — Install

Interactive install guide + download for the **AutoSortBins** Premiere Pro panel.

➡️ **Install page:** https://crockerfilm.github.io/autosortbins/

AutoSortBins sorts loose clips in your Project panel into named bins (Footage,
SFX, Music, Graphics, Sequences, …) and cleans up empty bins. It always previews
first and saves a project backup before making changes — it never deletes clips.

This is a pre-release sideload build for the team while the official Adobe
Exchange listing is in review. The guide is plain HTML and the kit is plain
files, so there's nothing for macOS/Windows security to block.

- `index.html` — the interactive, pick-your-OS install guide
- `AutoSortBins-Install.zip` — the plugin folder + registration file + instructions
